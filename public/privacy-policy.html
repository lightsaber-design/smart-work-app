Política de Privacidad

MinistryLog · Última actualización: 12 de junio de 2026

1. Responsable de los datos

Esta aplicación es desarrollada de forma individual por Marco Antonio Tévar Asensio. Contacto: marco.antonio.tevar.asensio@gmail.com

2. Datos que la app recopila

MinistryLog no recopila ni transmite datos personales a ningún servidor externo. Toda la información que el usuario introduce se almacena únicamente en el dispositivo:

  • Nombre del usuario (opcional, solo para personalización local)
  • Ciudad de referencia (nombre, latitud y longitud) para cálculos de horario solar
  • Registros de tiempo y actividades de ministerio
  • Eventos de calendario y sesiones de estudios bíblicos
  • Lugares favoritos marcados en el mapa
  • Preferencias de configuración (idioma, tema, notificaciones)

3. Permiso de ubicación

La app solicita acceso a la ubicación del dispositivo únicamente para:

  • Autocompletar la selección de ciudad durante la configuración inicial.
  • Centrar el mapa al añadir un lugar favorito.

Las coordenadas de ubicación nunca se envían fuera del dispositivo. El usuario puede conceder o revocar este permiso en cualquier momento desde Ajustes → Aplicaciones → MinistryLog → Permisos.

4. Conexión a Internet

La app utiliza Internet exclusivamente para cargar los mapas de calles (teselas de OpenStreetMap, proveedor: openstreetmap.org). Estas solicitudes no incluyen datos personales del usuario. El resto de la funcionalidad de la app es completamente offline.

5. Notificaciones

Las notificaciones locales (recordatorio de temporizador, objetivo mensual) se generan íntegramente en el dispositivo mediante el sistema de notificaciones de Android. No se utilizan servicios de push de terceros.

6. Copias de seguridad y exportación

El usuario puede exportar todos sus datos en formato JSON en cualquier momento desde la pantalla de Ajustes → Copia de seguridad. Estos ficheros permanecen bajo el control exclusivo del usuario. La copia de seguridad automática de Android está desactivada para esta app, de modo que los datos no se suben a Google Drive sin consentimiento explícito.

7. Datos de terceros

MinistryLog no integra SDKs de analítica, publicidad, redes sociales ni ningún otro servicio de seguimiento de usuarios.

8. Menores de edad

La app no está dirigida a menores de 13 años. No se recopilan datos de forma intencionada de ningún usuario.

9. Cambios en esta política

Cualquier modificación se publicará en esta misma página. El uso continuado de la app tras la publicación de cambios implica la aceptación de los mismos.

10. Contacto

Para cualquier consulta relacionada con la privacidad, escribe a: marco.antonio.tevar.asensio@gmail.com

Privacy Policy

MinistryLog · Last updated: June 12, 2026

1. Data Controller

This app is developed individually by Marco Antonio Tévar Asensio. Contact: marco.antonio.tevar.asensio@gmail.com

2. What data the app collects

MinistryLog does not collect or transmit personal data to any external server. All information entered by the user is stored on-device only:

  • User name (optional, for local personalization only)
  • Reference city (name, latitude, longitude) for solar schedule calculations
  • Time entries and ministry activity records
  • Calendar events and Bible study sessions
  • Favorite places marked on the map
  • App preferences (language, theme, notifications)

3. Location permission

The app requests device location access only to:

  • Auto-fill the city selection during initial setup.
  • Center the map when adding a favorite place.

Location coordinates are never sent outside the device. This permission can be granted or revoked at any time via Settings → Apps → MinistryLog → Permissions.

4. Internet access

The app uses the Internet exclusively to load street map tiles (OpenStreetMap, provider: openstreetmap.org). These requests do not include personal user data. All other app functionality works fully offline.

5. Notifications

Local notifications (timer reminder, monthly goal) are generated entirely on-device using the Android notification system. No third-party push services are used.

6. Backup and export

Users can export all their data as a JSON file at any time via Settings → Backup. These files remain under the user's exclusive control. Android's automatic cloud backup is disabled for this app, so data is not uploaded to Google Drive without explicit user action.

7. Third-party SDKs

MinistryLog does not include any analytics, advertising, social media, or user-tracking SDKs of any kind.

8. Children

This app is not directed at children under 13. No data is intentionally collected from any user.

9. Changes to this policy

Any changes will be published on this page. Continued use of the app after changes are posted constitutes acceptance of those changes.

10. Contact

For any privacy-related questions, write to: marco.antonio.tevar.asensio@gmail.com
